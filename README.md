#### Usage
##### Add dependencies
```groovy
implementation 'com.arch.mvp:arch-mvp:1.0.4'
```

##### In you `build.gradle` add setting
```groovy
    compileOptions {
        sourceCompatibility JavaVersion.VERSION_1_8
        targetCompatibility JavaVersion.VERSION_1_8
    }
```
